# D4 Gear Calculator — 产品需求文档

**最后更新：2026-07-12**

---

## 一、项目概述

暗黑4（PC）装备提升计算器。手动输入当前套装每件装备的词缀数值，估算换装后伤害提升/降低的百分比。

**核心原则：**
- 纯手动输入，不读取游戏窗口、不做 overlay、不碰游戏内存
- 单个 HTML 文件，本地浏览器运行，无需服务器
- 数据存储在 localStorage，支持导出/导入 JSON 备份

---

## 二、伤害计算模型

基于「憎恨之王」版本重做后的乘区结构：

```
基础伤害 = 技能伤害倍率 × 主属性倍率 × A桶倍率 × B桶倍率 × C桶倍率

期望伤害 = 基础伤害 × (1 + 暴击率 × 暴击伤害加成)

攻速：独立显示，不混入伤害乘区
```

### 五个乘区

| 乘区 | 来源 | 计算方式 |
|------|------|----------|
| 技能伤害倍率 | 技能专属词缀（如"+X% 旋风斩伤害"） | 每条独立相乘 |
| 主属性倍率 | 力量/敏捷/智慧/意志 | 所有点数加总，每点=0.1%增伤，换算为 `1 + 总点数×0.001` |
| A桶 | 巅峰盘数值 + 装备"+"类词缀 | 所有来源加总后 `(1 + ΣA%)` 作为一个因子（不区分子类型，全部直接相加） |
| B桶 | "前置*"词缀，含4个类别：暴击伤害、易伤伤害、元素伤害、持续伤害 | 同名类别相加合并，不同类别之间独立相乘 |
| C桶 | "后置*"词缀，独立乘区 | 每一条单独相乘，同名也不合并 |

### B桶内的"适用范围"（scope）与 暴击/持续伤害 双路径

B桶内每个类别有一个适用范围：`both`（两条路径共享，如易伤伤害、元素伤害）、`crit`（只喂暴击路径，即暴击伤害）、`dot`（只喂持续伤害路径，即持续伤害）。

暴击伤害、持续伤害虽然各自只影响一条路径，但在数据结构上仍然是B桶的成员（同名相加），只是不参与B桶的"跨类别相乘"，而是分别用于各自路径的公式：

```
基础伤害（共享） = 技能伤害倍率 × 主属性倍率 × A桶倍率 × B桶倍率(仅both类别) × C桶倍率

直接伤害期望 = 基础伤害 × (1 + min(暴击率, 100%) × 暴击伤害%)
持续伤害期望 = 基础伤害 × (1 + 持续伤害%)
```

- 暴击率 = 全局基础暴击率（角色自带）+ 所有装备词缀中「暴击率」的加总，只影响直接伤害路径
- 暴击伤害（B桶，scope=crit）：只影响直接伤害路径，不影响持续伤害
- 持续伤害（B桶，scope=dot，如出血/毒/燃烧持续跳伤）：只影响持续伤害路径，不受暴击率/暴击伤害影响，也不能暴击
- 易伤伤害、元素伤害（B桶，scope=both）：两条路径都共享
- 自定义B桶词缀可手动选择 scope（两者共享/仅暴击/仅持续）

### 攻速

- 从所有词缀中汇总「攻击速度」加成
- 显示攻速倍率 = `1 + 总攻速%`
- 独立展示，不影响伤害乘区

## 三、功能需求

### 3.1 装备槽位

10个槽位，对应D4装备位置：

| 槽位 | 中文名 | 图标 |
|------|--------|------|
| helm | 头盔 | ⛑ |
| chest | 胸甲 | 🥋 |
| gloves | 手套 | 🧤 |
| pants | 裤子 | 🩲 |
| boots | 靴子 | 👢 |
| weapon | 武器 | ⚔ |
| offhand | 副手 | 🛡 |
| ring1 | 戒指1 | 💍 |
| ring2 | 戒指2 | 💍 |
| amulet | 护符 | 📿 |

每个槽位可添加任意数量词缀条目。

### 3.2 词缀输入

- 每条词缀：选择预设类型 + 填入数值
- 支持自定义词缀：填写名称，手动选择归属乘区（A/B/C桶等）
- 词缀按桶类型颜色区分（见配色规范）

### 3.3 对比候选装备

- 选中某个槽位后，点击「对比候选」按钮
- 中间编辑区变为左右两列：当前装备 | 候选装备
- 系统自动复制当前槽位词缀到候选列，可独立编辑
- 其余9件装备保持不变
- 右侧面板实时显示各乘区变化和期望伤害提升%

### 3.4 全局输入

| 字段 | 说明 | 默认值 |
|------|------|--------|
| 基础暴击率 | 角色自带暴击（不来自装备） | 5% |
| 额外技能倍率 | 来自天赋/专精的技能加成（不好放入槽位的部分） | 0% |

### 3.5 结果面板

**普通模式：**
- 各乘区倍率（技能、主属性、A桶、B桶、C桶）
- 暴击率、暴击伤害汇总值
- 基础伤害倍率
- 期望伤害倍率（含暴击）
- 攻速倍率

**对比模式：**
- 各乘区变化（当前→候选，%差值）
- 当前 vs 候选期望伤害绝对值
- 期望伤害变化（大字突出显示，正绿负红）
- 攻速变化

### 3.6 数据持久化

- **自动保存**：每次修改后自动写入 localStorage（key: `d4calc_v2`）
- **导出**：下载 JSON 文件（`d4-gear.json`）
- **导入**：选择本地 JSON 文件恢复数据
- **清空**：二次确认后清除所有槽位数据

## 四、预设词缀列表

### 特殊属性

| 预设ID | 显示名 | 归属 |
|--------|--------|------|
| mainstat | 主属性 | 主属性桶（自动换算） |
| skill_dmg | 技能伤害 | 技能桶（各自独立×） |
| crit_rate | 暴击率 | 暴击公式（概率） |
| atk_spd | 攻击速度 | 独立显示 |

### A桶（单一预设，纯加法）

`a_dmg` A桶伤害(+) —— 所有来源在机制上本就直接相加，不需要区分子类型，因此只保留一个预设入口

### B桶（4个类别，同类加总、异类相乘）

| 预设ID | 显示名 | scope |
|--------|--------|-------|
| b_crit | 暴击伤害 | crit（仅暴击路径） |
| b_vul | 易伤伤害 | both（两路径共享） |
| b_elemental | 元素伤害 | both（两路径共享，涵盖火/冰/闪电/毒/暗影/物理等，按自己build用到的类型填一个即可） |
| b_dot | 持续伤害 | dot（仅持续路径） |

精英伤害、超载伤害、近战/远程范围伤害、出血/燃烧/击晕伤害等细分类别不再单独预设，可通过「自定义词缀」手动添加并选择归属B桶 + scope。

### C桶（每条独立相乘）

传奇词条加成、独特词条加成、自定义C桶词缀
